# TodoList

## Installation
```
npm install
```
## Configuration

### Hardhat
```
cp hardhat.config.example.js hardhat.config.js
```

### TopoList
```
cp src/config.example.json src/config.json
```


## Hardhat
### Compile
```
npx hardhat compile
```
### Deploy
```
npx hardhat run backend/scripts/deploy.js
```

## Run
```
npm run start
```
